# Saleae Analyzer

Saleae Logic 2 Extension to simplify decoding DroneCAN messages.

## Usage

#### Importing Extension

To import extension into logic,

1. Click on 'Extensions' in the right toolbar
2. Press the menu button on the right side of the title and click on 'Load Existing Extension'.
3. Now navigate to the saleale-analyzer folder and import the extension.json file.

#### Using Extension

With Saleae Logic open,

1. Click on 'Analyzers' on the right toolbar
2. Click the + button on the toolbar that pops up and then click on 'DroneCAN_HLA'.
3. Choose its input analyzer and then press Save to use the extension.

Wait for a few seconds and then DroneCAN_HLA Channel should populate.
