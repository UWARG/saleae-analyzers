# Saleae Analyzer

Saleae Logic 2 Extension to simplify decoding DroneCAN messages.

## Usage

### Importing Extension

To import extension into logic click on extensions in the right toolbar, press the three-buttons and click on 'Load Existing Extension'. Now navigate to the saleale-analyzer folder and import the extension.json file.

### Using Extension

With Saleae Logic open, click on 'Analyzers', then the + button on the toolbar that pops up and then click on 'DroneCAN_HLA'. Choose its input analyzer and then press Save to use the extension. Wait for a few seconds and then DroneCAN_HLA Channel should populate.
